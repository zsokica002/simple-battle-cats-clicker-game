# simple-battle-cats-clicker-game
Simple, fan-made, "The Battle Cats" inspired clicker game to test my beginner JS coding skills. Made in React.

WORK IN PROGRESS - as I hone my skill, I will be adding to this project. When I am happy with the results, I will declare it officially finished, and any further update will be a new version after the fact.

During my frontend programming course, I got interested in simple animations in CSS. I wanted to test my newfound skill and I made a short jumping animation of a "Cat" from the tower defense mobile game "The Battle Cats". Then I decided I wanted to make it more functional so I added a simple "clicker game" function.

Excuse my code, I am a beginner and still learning. I am only uploading it here to showcase my passion for and interest in game development, despite being a newbie still.

Enjoy!
